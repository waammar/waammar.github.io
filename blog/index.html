There has been a lot of those guides but here’s another one to the collection with minor improvements. In this tutorial we will install step by step the Nginx admin cPanel plugin.

Installation for 32Bit/64Bit

Step 1 Install Nginx Admin

    cd /usr/local/src
    wget http://nginxcp.com/latest/nginxadmin.tar
    tar xf nginxadmin.tar
    cd publicnginx
    ./nginxinstaller install
